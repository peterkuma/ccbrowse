About

ccbrowse is an open source web application for browsing data from the CALIPSO and CloudSat satellites.

How to use

For navigation, press and drag the image. Zoom by scrolling. Read off values by double-clicking. Select UTC time with the blue progress bar above.

Data Quality

Data quality is not guaranteed. Especially,

  • The quality of interpolation is limited.
  • Quality Control (QC) information is not shown.

Interpolation

Data is interpolated by averaging all data points overlying a given pixel of the resulting image. When the scale of the image is greater than that of the data, ordinary nearest-neighbor interpolation is performed.

More Information

For more information, please visit ccbrowse on GitHub.
